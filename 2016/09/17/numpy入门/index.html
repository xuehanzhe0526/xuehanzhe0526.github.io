Data Types

Each value in a NumPy array has to have the same data type。

NumPy数组中的值 必须具有相同的数据类型。NumPy的数据类型 与python的数据类型是相似的

  • bool – Boolean. Can be True or False.

  • int – Integer values. An example is 10. Can be int16, int32, or int64. The suffix 16, 32, or 64 indicates how long the number can be, in bytes (we’ll dive more into bytes later on). The larger the suffix, the larger the integers can be.

  • float – Floating point values. An example is 10.6. Can be float16, float32, or float64. The suffix 16, 32, or 64 indicates how many numbers after the decimal point the number can have. The larger the suffix, the more precise the float can be.

  • string – String values. Can be string or unicode. We’ll get more into what unicode is later on, but the difference between string and unicode is how the characters are stored by the computer.

using the dtype property 可以查看数据的类型

Inspecting The Data

world_alcohol_dtype = world_alcohol.dtype

world_alcohol数据的类型 是 float 这是因为 NumPy数组中的值 必须具有相同的数据类型。NumPy尝试着去转换数据类型 为 float 。genfromtxt()方法将试图猜出正确数据类型去创建数组（有时候并不正确）

而 我们的数据中 ，WHO Region, Country, and Beverage Types 这些列的真实值为字符串 。并不能转换为 float数据类型。当Numpy 不能转换值为 数值型类型（like float or integer）时，它将使用一个特殊的字符 nan用来表示该值 不是数值型数据。

在数据缺失时，Numpy会使用另外 一个特殊字符na

nan and na values are types of missing data.

在world_alcohol.csv文件中，首行表示每一列的名称，并不是真实数据的一部分，而且是字符串类型，由于字符串不能转换为float类型，所以Numpy使用nan去代替它们。

在上面的图片中 出现类似于这样的数值1.98600000e+03
1.98600000e+03 的真实值为 1.986 * 10 ^ 3, or 1986

1000000000000000 can be written as 1e+15
