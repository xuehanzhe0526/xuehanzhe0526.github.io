Scopes

一个全局范围global scope, 一个局部范围local scope

  • There’s only one global scope, but each function creates its own local scope.

Scope Isolation

两个独立作用域 ，虽然他们的名字是一样的。当一个函数被调用后，它的本地变量total就已经释放了。

Scope Inheritance

当使用一个变量，然而在本地没有定义python编译器会接着查找 global scope

Inheritance Limits

我们可以在局部范围去访问一个 global scope，但是在局部范围不能去改变 global scope的值，否则就会报错。

Built-In Inheritance

如果局部变量 并没有定义 在local scope，python解释器将会在全局范围（ global scope）中寻找，若没有，它会查看是否为内置函数。
