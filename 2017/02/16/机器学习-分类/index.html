Feb 16, 2017

线性回归 Linear regression 的目标列 target column 是一个有序、连续的数值 （is ordered and continuous）
当目标列 包含离散的数值 discrete values 线性回归将不适合。
这样的问题成为：分类问题 classification problems

可以发现 我们要从 Categories 转变为 Numerical Categories

这节，我们只关注 binary classification 二值分类

数据（ 大学生录取预测）

https://github.com/ntandresen/dataquest/blob/master/admissions.csv

这是原本的数据，admit的值是0或者1。可以发现”gpa”和”admit”并没有线性关系，因为”admit”只取两个值。

这是通过线性回归模型预测的admit的值，发现admit_prediction 取值范围较大，有负值，不是我们想要的。

Logistic Regression 逻辑回归

逻辑回归是一个流行的分类方法,它将输出限制在0和1之间（非负数）。这个输出可以被视为一个给定一组输入某个事件的概率,就像任何其他分类方法

线性回归 函数： y = mx + b

Accuracy 准确率

What fraction of the predictions were correct (actual label matched predicted label)?


在逻辑回归中，使用鉴别阀值 discrimination threshold ，scikit-learn默认使用0.5 如果预测概率大于0.5 观察标签设置为1 反之为0

###

Sensitivity 灵敏度／真阳性率

灵敏度（Sensitivity，也称为真阳性率）是指实际为阳性的样本中，判断为阳性的比例

（例如真正有生病的人中，被医院判断为有生病者的比例）

How effective is this model at identifying positive outcomes?

如果 这个值 很低 意味着 模型 catching positive cases（标签为1 应该被录取的学生） 效率低

我们期望获得一个高的 sensitive 模型 如果利用构造的模型去预测 一个病人是否得了癌症 cancer 如果被 病没有被 预测出 意味着 有可能病人失去生命。所以需要一个高的sensitive 模型能够 “catch“所有的 positive 情况 （the positive case 是病人with 癌症）

特异度（Specificity，也称为真阴性率）

是指实际为阴性的样本中，判断为阴性的比例
（例如真正未生病的人中，被医院判断为未生病者的比例）
这里是 ，申请人被正确地拒绝的比例。

如何有效的 识别出negative 结果。

高的特异度（Specificity）意味着 模型擅长预测 那些申请人 应该被拒绝。

任何测试而言，都需要在灵敏度及特异度之间进行取舍
